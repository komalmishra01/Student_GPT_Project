Yah ek Python-based desktop application hai jo CustomTkinter aur Google Gemini API ka upyog karke alag-alag academic modes mein students ko madad karti hai. Is project mein Front-end (UI), Back-end (AI Logic), aur Data Management ko alag-alag modules mein rakha gaya hai.

---

## ‚ú® Features

* **Modular Design:** Front-end (`app.py`), Back-end (`ai_core.py`), aur Data Management (`data_manager.py`) files alag-alag hain.
* **Multiple Modes:** User apni zaroorat ke hisaab se AI ka mode badal sakta hai (jaise: Maths Expert, Creative Writing Coach).
* **Session History:** Chat history JSON file (`chat_history.json`) mein automatic save aur load hoti hai.
* **Secure API Handling:** API Key ko `.env` file ya Environment Variable ke through surakshit tareeke se load kiya jaata hai.
